# generator-react
> React generators

## Installation

First, install [Yeoman](http://yeoman.io) using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
```

To generate a new component:

```bash
yo react:component
```

To generate a new page:

```bash
yo react:page
```
